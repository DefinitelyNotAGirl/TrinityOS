1 Terms and Definitions

1.1 Numbers

Hexadecimal (base 16) numbers are marked using the prefix “0x”, likewise, base 4 numbers are marked with “0q”, base 8 numbers with “0o”, binary numbers with “0b” and decimal numbers are marked with “0d”, any numbers not marked with one of these prefixes is to be treated as decimal (base 10). Furthermore, all numbers are to be treated as big endian unless indicated otherwise.

1.3 Access Types

MnemonicDescription
R/WThe field can be written to and read from as normal
R/OThe field can be read from, but cannot (or should not) be written to, do not rely on the underlying memory being read-only
MBZThe field must always have all its bits cleared to 0
MBOThe field must always have all its bits set to 1
N/AThe field cannot be accessed
